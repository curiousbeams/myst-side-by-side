# Overview & basic usage

The `{side-by-side}` directive renders two versions of a document — an
original and its translation — in linked, scrollable panels. It is implemented
as a MyST plugin (`side-by-side.mjs`: one directive plus two transforms) and an
AnyWidget client module (`side_by_side_widget.mjs`) that provides the
interactive behavior.

## Usage

Point the directive at the two documents, exactly like `{include}`:

```markdown
:::{side-by-side}
:old: ./old/01.snippet_example_old.md
:new: ./new/01.snippet_example_new.md
:::
```

## The `.old` / `.new` contract

The author links the two documents by giving corresponding passages labels
that share a stem: a paragraph labeled `(opening_old)=` in the original pairs
with `(opening_new)=` in the translation (the dot forms `opening.old` /
`opening.new` work too). The documents do *not* need the same length,
paragraph count, or structure — only the hooks matter.

The same contract drives equation, figure, and footnote numbering — see @content_page.

Hooks that lack a counterpart produce a **build-time warning**, so a
translation that drifts out of sync with its original is caught early.

## Try it

Click any passage below and the *other* panel scrolls to the nearest matching
hook and briefly flashes it — for example, click "the age of wisdom" on the
left, or « l'an de grâce » on the right. Also try:

- **drag** the divider between the panels to change the split;
- **double-click** the divider to reset it;
- **minimize** either panel with the `–` button in its header (the title turns
  vertical; `+` restores it);
- **scroll** each panel independently — the panels are capped at
  `:max-height:` and get their own scrollbars.

:::{side-by-side}
:old: ./old/01.snippet_example_old.md
:new: ./new/01.snippet_example_new.md
:max-height: 450px
:old-title: English — A Tale of Two Cities
:new-title: Français — Un conte de deux villes
:::

## Options

| Option                    | Default                     | Description                                                             |
| ------------------------- | --------------------------- | ----------------------------------------------------------------------- |
| `old` (required)          | —                           | Path to the original document, relative to the page (like `{include}`)  |
| `new` (required)          | —                           | Path to the translated document                                          |
| `max-height`              | `none`                      | Panel height cap (any CSS length); use `none` to let the panels grow     |
| `split`                   | `50%`                       | Initial width of the left (old) panel; the divider stays draggable       |
| `highlight`               | `true`                      | Flash the paired block after a click-to-jump                             |
| `old-title` / `new-title` | the file names              | Panel header titles                                                      |
| `widget`                  | `./side_by_side_widget.mjs` | Path to the AnyWidget ESM module, relative to the page                   |
| `class`                   | —                           | Extra CSS classes for the container                                      |

Snake-case aliases are accepted for compatibility (`max_height`, `old_title`,
`new_title`, and `proportional_width` for `split`).
